# Survive Game

## Build Setup

```bash
# Necessário instalação do NodeJS
# Dentro da pasta do projeto, para instalação das dependências:
$ npm install

# Servidor local localhost:3000
$ npm run dev

# build for production and launch server
$ npm run build
$ npm run start

# generate static project
$ npm run generate
```
